# 🚇 Metro Ticket Generator using ServiceNow

## 📌 Overview

The **Metro Ticket Generator** is a ServiceNow-based application that automates the metro ticket booking process using the **Service Catalog** and **Flow Designer**. It enables users to book metro tickets by selecting the source station, destination station, travel date, ticket type, and number of passengers through a simple and user-friendly interface.

Once the booking request is submitted, the application automatically processes the request, validates the booking details, and generates a confirmation. This project demonstrates workflow automation in ServiceNow and provides a streamlined ticket booking experience.

* Metro Ticket Booking through Service Catalog
* User-friendly booking form

* Travel date selection
* Ticket type selection

* Automated booking workflow using Flow Designer
* Metro station details management

---

## 📋 Project Workflow

1. Create the Metro Ticket Booking Service Catalog.
2. Design the Metro Ticket Booking Catalog Item.
3. Configure catalog variables for passenger and travel information.
4. Store metro station details in a custom table.
5. Trigger a Flow Designer workflow when a booking request is submitted.
6. Retrieve the submitted catalog variables.
7. Process the booking request.
8. Generate the booking confirmation.
9. Display the generated ticket details to the user.
